Welcome to

DemocrApp [RC1]

This site provides online voting functionality for general meetings. In order to access the site you will need to use the anonymous access token that you were given when you registered into the meeting by the meeting secretary.

If you have not yet registered, please see the company secretary/meeting chair to request a token.

FAQs:
  • What if I leave? You can only access DemocrApp from within the Bedlam WiFi network. This site will stop working once you leave Bedlam's WiFi range.
  • What if my device doesn't work properly? Take your token to one of the e-voting kiosks and place your votes there.
  • What if my device stops working? You can re-authenticate on any compatible device, or move to using one of the aforementioned e-voting kiosks.

To help us squash any bugs, if your device encounters an error while running DemocrApp, it will send a an anonymous report to Sentry with a little more info about what happened.
